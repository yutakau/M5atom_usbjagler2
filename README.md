# M5atom_usbjagler2
マウスジグラーと１キーBLEキーボードの複合デバイス



　BLEキーボード機能
　　ボタン(G39)を押すとあらかじめプログラムされた特定の文字列をキー入力する。
　　この時、マトリクスLEDが1秒間赤色になる。

　BLEマウス機能
　　２分に１回、スクリーンロックを外す目的でマウスを移動させる。上下左右にマウスカーソルを10cnt移動させる。
　　この時、マトリクスLEDが1秒間青色になる。
　　この機能は外部ピンG33をGNDに接続すると無効にできる。

・マトリクスLED25個がある。上記機能が動作していないときは、適宜、時間変化するグラデーション表示する。

・M5Stack のM5Atomを使用。
・ライブラリはFastLEDと、適宜BLEマウス、BLEキーボードのライブラリを使用。
・開発環境はPlatform IDEでAruduinoを使用。
